Logs an event to a file with a timestamp and actuator data.

This function logs an event by appending it to a log file. It records the event ID, the timestamp (in milliseconds), and actuator states in a structured format.

Parameters
id_aebIdentifier for the AEB system.
event_idThe unique event identifier (formatted as a hexadecimal string).
actuatorsStructure containing actuator state information.

Note
This function is a called from the actuators module, and it is used as a way to guarantee tava every change to the AEB internal state is captured and stored on the log file.
