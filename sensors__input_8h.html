Detailed Description

Defines the data structure for sensor inputs to the AEB system.

This file declares the structure that holds sensor input values required by the AEB (Autonomous Emergency Braking) system. These values represent real-time data from vehicle sensors used for decision-making.

  • Contains relative velocity and acceleration between vehicle and detected object.
  • Flags for obstacle presence and system activation status.
  • Pedal input statuses from driver (brake and accelerator).
  • Contais gear information - if the vehicle is in reverse or forward mode
